#  University Data Dashboard

Dashboard interactivo desarrollado con **Streamlit**, basado en el dataset *university_student_data.csv*.

##  Descripción
Este proyecto forma parte de la actividad **Data Visualization and Dashboard Deployment**.  
El objetivo es analizar indicadores universitarios de retención, satisfacción y matrícula, y mostrarlos en un tablero dinámico.

##  Funcionalidades
- Filtros por año y término académico (Spring/Fall)  
- Indicadores KPI: Applications, Retention Rate, Student Satisfaction  
- Visualizaciones:
  1. Tendencia del Retention Rate (%) por año  
  2. Student Satisfaction (%) promedio por año  
  3. Comparación Spring vs Fall  
- Tabla dinámica con los datos filtrados

## Tecnologías usadas
- Python  
- Pandas  
- Matplotlib / Seaborn  
- Streamlit  

## Ejecución local
Para ejecutar la aplicación en tu computadora:

```bash
pip install -r requirements.txt
streamlit run app.py
```
